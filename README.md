# Jikan Anime

## VariantCode: JIKAN-ANIME-MOD_A5_SKELETON

Приложение для поиска и просмотра информации об аниме с использованием Jikan API.

## Использованные endpoints

**Базовый URL:** https://api.jikan.moe/v4/

| Endpoint | Описание |
|----------|----------|
| `GET /anime?q={query}&page=1` | Поиск аниме по названию (список) |
| `GET /anime/{id}` | Получение детальной информации об аниме |

### Основной модификатор (по варианту): **MOD_A5_SKELETON**
Вместо стандартного индикатора загрузки (`CircularProgressIndicator`) на экране списка используется **скелетон**:
- Отображаются плейсхолдеры изображений и текста
- Плавный переход к загруженному контенту

### Дополнительные модификаторы (реализованы сверх задания для красоты)

#### **MOD_B8_SHARE_DETAIL**
На экране деталей добавлена кнопка "Поделиться", которая открывает системный Intent для отправки информации об аниме.

#### **MOD_E25_FAV_SORT**
На экране избранного реализована сортировка:
- По названию
- По рейтингу
- По году выпуска
- По дате добавления

## Технологический стек

- **Язык:** Kotlin
- **UI:** Jetpack Compose + Material 3
- **Асинхронность:** Coroutines + Flow
- **Сеть:** Retrofit + OkHttp
- **DI:** Dagger Hilt
- **Архитектура:** Clean Architecture (Data → Domain → Presentation)
- **Навигация:** Navigation Compose
- **Загрузка изображений:** Coil
- **Скелетон:** placeholder
